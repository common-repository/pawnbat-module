This plugin gives a very simple way to integrate PawnBat Module into your WordPress site.

== Installation ==
1. Upload the plugin to the '/wp-content/plugins/' directory.
2. Activate it through the 'Plugins' menu in WordPress.
3. Enable the plugin and enter Pawn Shop ID: Settings -> PawnBat Module
